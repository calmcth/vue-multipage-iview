# 多页应用 webpack3-vue-vuex-iview+国际化

## 全局需要安装的环境

```
 nodejs
 
 npm install cnpm -g

 npm install webpack -g

 npm install typescript -g
 
```
## 使用

```
npm install 

```
## 编译

```
npm run build 

```

## 运行 

```
 npm run dev 
 
```
## 启动后的地址说明
```
对应的地址为build/utils/ 内的name值
该系统有2个地址 http://127.0.0.1:9090/client.html,http://127.0.0.1:9090/admin.html
 
```
